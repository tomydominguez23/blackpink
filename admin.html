Asistente de inventario y ventas

¿En qué puedo ayudarte?

Preguntá por stock, un IMEI concreto, pedidos o pedí un resumen. Las respuestas se arman con consultas de solo lectura a tu Supabase.

Si está desactivado, solo se usan reglas locales. Si OpenAI no está configurado en el servidor, se responde con reglas automáticamente.

✦

Con IA avanzada, OpenAI elige qué consultas hacer (solo lectura) vía la Edge Function assistant-query. Sin API key en Supabase, el panel usa solo reglas.

Configuración General

Se usa en el panel de control para marcar productos publicados con poco stock. Se guarda en este navegador.

WhatsApp no está integrado en el servidor: podés usar el enlace wa.me en el cuerpo del mail o un proveedor aparte. Configurá en Supabase los secrets RESEND_API_KEY, NOTIFY_FROM_EMAIL, NOTIFY_TO_EMAIL.

Exportar CSV

Archivos compatibles con Excel (UTF-8). Los pedidos requieren la migración SQL de permisos en Supabase.
